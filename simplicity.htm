HTML over CSS

Vendor specific layout engines have specific CSS attributes.

EdgeHTML - Microsoft Edge.

Blink - Google Chrome/Chromium and Opera.

Gecko - Mozilla Firefox.

WebKit - Safari/Konqueror.

Trident - Internet Explorer.

KHTML - Konqueror (Discontinued).

Presto - Opera (Discontinued).

Amount of properties.

There are approximately 228 CSS properties.

Whereas there are only approximately only 170 HTML style attributes.


Although CSS is very useful for styling, try and use HTML attributes over CSS attributes for compatability and lightweight reasons.

Personal example: I could of added 'font-size: 120%;' to the b tag property in my CSS style sheet. But it wasn't necessary.

So I stuck with just 'color: #ffff00;'.
